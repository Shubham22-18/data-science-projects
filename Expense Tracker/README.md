# Expense Tracker

A simple Expense Tracker application built using Python. This project helps users record and manage daily expenses efficiently through a command-line interface.

## Features
- Add expenses
- View all expense records
- Calculate total expenses
- Save records permanently using JSON file handling

## Technologies Used
- Python
- JSON
- File Handling

## Concepts Used
- Lists
- Dictionaries
- Loops
- Conditional Statements
- Functions
- File Handling

## How to Run

1. Save the file as:
   expense_tracker.py

2. Open terminal or command prompt.

3. Run the program:

   python expense_tracker.py
