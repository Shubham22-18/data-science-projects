# Student Management System

A beginner-friendly Python project to manage student records using a simple command-line interface.

## Features
- Add student records
- View all students
- Search student by roll number
- Delete student records

## Technologies Used
- Python

## How to Run
1. Save the file as:
   student_management_system.py

2. Run the program:
   python student_management_system.py

## Purpose
This project helps beginners understand:
- Dictionaries
- Loops
- Conditional statements
- User input handling
