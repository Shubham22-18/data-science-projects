## Projects Included

### 1. Calculator
A basic Python calculator for performing arithmetic operations such as:
- Addition
- Subtraction
- Multiplication
- Division

### 2. Weather App
A weather application that fetches real-time weather information using API integration.

**Features**
- Get city weather
- Temperature details
- Weather condition information

### 3. Student Management System
A menu-driven Python application to manage student records.

**Features**
- Add student
- View student records
- Search student
- Delete student

### 4. To-Do App
A task management application built using Python.

**Features**
- Add tasks
- View tasks
- Remove tasks

### 5. Number Guessing Game
A fun Python game where users guess a randomly generated number.

**Features**
- Random number generation
- Guess hints
- Attempt tracking

### 6. Expense Tracker
A Python-based expense management application.

**Features**
- Add expenses
- View expense records
- Calculate total expenses
- Save records using file handling
