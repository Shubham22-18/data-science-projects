# Number Guessing Game

A fun Python game where the player tries to guess a randomly generated number between 1 and 100.

## Features
- Random number generation
- Unlimited guessing attempts
- Hints for high and low guesses
- Tracks number of attempts

## Technologies Used
- Python
- Random Module

## How to Run
1. Save the file as:
   number_guessing_game.py

2. Run the program:
   python number_guessing_game.py

## Objective
The player must guess the correct number with the help of hints provided by the program.
