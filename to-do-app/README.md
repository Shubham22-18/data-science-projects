# To-Do App

A simple command-line To-Do application built using Python to manage daily tasks efficiently.

## Features
- Add tasks
- View task list
- Remove completed tasks
- Simple menu-driven interface

## Technologies Used
- Python

## How to Run
1. Save the file as:
   todo_app.py

2. Run the program:
   python todo_app.py

## Learning Outcomes
This project demonstrates:
- Lists in Python
- Loops
- User interaction
- Basic task management logic
